【此为Web前端构建工具 viteJS 生成的 Vanilla 框架】


运行方法：
1. 打开 VScode，顶部新建 New Terminal
2. 确保当前路径是所在文件夹的路径
3. 打开界面：npm run dev
4. 终止界面：ctr+c


编辑方法：
1. index.html 是首页文档，<head>部分引入组件css，<body>部分可以看到很多<section>存放不同的组件，<script>部分引入组件 html 和 js
2. 我们只需要在 src/components/ 文件夹下找到对应的组件文件夹进行编辑，文件夹里包含三类文件：html（界面）、css（样式）、js（交互函数）
3. 如果要修改文件名，记得全局修改！！！！！


【养成好习惯：开始编辑前 git pull，完成编辑后 git push】
